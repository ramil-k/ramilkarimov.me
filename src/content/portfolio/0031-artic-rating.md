Frontend assessment task I designed for evaluating developer candidates. A React application that fetches artworks from the Art Institute of Chicago API, displaying them in a card grid with detail pages and a half-star rating system persisted to LocalStorage.

Features custom React hooks for data fetching and state management, accessible rating controls with ARIA labels, and loading/error states. Built with Vite and TypeScript, demonstrating clean component architecture and API integration patterns.
