Native time tracking app for iOS and Mac that stores data locally in human-readable files, keeping you in control of your data. Uses Automerge CRDT for conflict-free sync across devices via any folder sync service (Dropbox, iCloud Drive, Google Drive).

Key features include multiple concurrent timers, tag-based organization with deterministic OKLCH-generated colors, CSV import/export, iOS Shortcuts integration, home screen widgets, and a unified codebase that runs on iPhone, iPad, and Mac via Catalyst.
