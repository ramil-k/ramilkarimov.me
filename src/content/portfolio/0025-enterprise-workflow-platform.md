# Full-stack development for enterprise workflow platform

Working on a configurable enterprise platform that enables organizations to build custom workflow applications without traditional coding.

## Role

Full-stack developer working across the Java backend and JavaScript frontend layers.

## Technologies

- **Backend:** Java, Maven, Jetty, Oracle Database
- **Frontend:** JavaScript (ES6), CSS
- **Testing:** Selenium, TestNG, Playwright
- **CI/CD:** GitHub Actions
- **Tools:** Git worktrees for parallel development
